# WApp - Wrapped App Mobile

A simple cordova app based on jQuery Mobile.

## Build

- `git clone https://github.com/giosil/wapp.git`
- `npm install -g cordova`
- `cordova add android`
- `cordova build`

## Run

- `cordova run android --device`
